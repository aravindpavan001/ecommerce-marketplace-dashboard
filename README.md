Ecommerce Marketplace Performance Dashboard

   Project Overview

This project analyses an ecommerce marketplace dataset using Power BI.
The dashboard provides insights into sales performance, customer behaviour, product categories, and seller performance.

   Tools Used

. SQL Server – Data storage and querying
. Power BI – Data visualisation and dashboard creation
. CSV Files – Raw dataset format
. GitHub – Project documentation and version control

   Key Metrics

. Total Sales
. Total Orders
. Total Customers
. Average Order Value (AOV)

   Dashboard Insights

. Credit cards dominate payments (~74% of transactions)
. São Paulo drives the highest order volume
. Beauty & Health is the top revenue-generating category
. Revenue is concentrated among a small number of sellers

   Dashboard Features

. Interactive filters (Year and State)
. Monthly order trend analysis
. Top product categories by sales
. Payment method distribution
. Top performing states
. Top sellers by revenue

   Dataset Source

Olist Ecommerce Dataset (Kaggle)
